Graph is truth

docs / chunks / nodes / edges / commits in .glane/. BM25 and vectors are rebuildable caches — delete them anytime.

Deterministic first

Markdown, code, and JSON extractors with provenance. No LLM key required for day-one value.

Context packs

glane context seeds search, expands the graph, and packs citations under a token budget — the verb agents run before work.

Versioned knowledge

Every sync/commit/forget is a generation. log, diff, and honest forget tombstones included.

HTTP + MCP

glane serve exposes /v1/search, /v1/context, /v1/query, and POST /mcp tools on a sequential race-safe loop.

Agent-native CLI

JSON stdout, typed errors, exits 80–119, embedded guide + help-json. Complies with cli-specs.

One binary

~240 KB native (MFL/machin). No Docker, Postgres, or vector DB to boot.

Self-hosted only

MIT, feature-complete on your machine. No hosted cloud product — see ROADMAP.

Install

curl -fsSL https://github.com/javimosch/glane/releases/latest/download/glane -o glane
chmod +x glane && ./glane guide --human

Linux x86_64. Or build from source with machin: ./build.sh.

The 60-second tour

glane search "error handling" --limit 5
glane query neighbors doc:README.md --dir out
glane log --limit 5
glane diff
glane forget path/to/stale.md
glane embed --limit 10          # optional, localhost only
glane bench --docs 1000
